# Tài nguyên công khai (Public Assets)

Thư mục này chứa các file tĩnh (hình ảnh, icon) được phục vụ trực tiếp bởi Web Server.

## Hướng dẫn thêm Logo:
1. Chuẩn bị file hình ảnh logo của phòng khám.
2. Đổi tên file thành **`logo.png`**.
3. Kéo thả (Upload) file `logo.png` vào thư mục này.
4. Sau khi upload, logo sẽ tự động hiển thị trên header của ứng dụng.

*Lưu ý: Không xóa thư mục này.*
